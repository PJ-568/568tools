## Introduction

Sample project using [gulp](http://gulpjs.com/).

gulp plugins used:

- gulp-coffee
- gulp-concat
- gulp-ugilfy
- gulp-ruby-sass

Get more info in [gulpfile.js](gulpfile.js)

## Build locally

```
// Install dependencies
npm install
// Run task with gulp
npm run build
```
